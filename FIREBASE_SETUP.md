# Firebase Setup (Настройка Firebase)

Чтобы сбор данных работал централизованно, вам необходимо настроить проект Firebase:

1. **Создайте Firebase проект** на [console.firebase.google.com](https://console.firebase.google.com/)
2. **Добавьте веб-приложение** и скопируйте ключи конфигурации.
3. **Включите Firestore Database** в режиме тестирования.
4. **Обновите `js/config.js`**, вставив свои ключи.

Это позволит админ-панели видеть ответы от всех пользователей в реальном времени.
